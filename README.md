# Pulse - Mini Social Media Platform

Pulse is a modern mini social media application built using HTML, CSS, and JavaScript. The platform allows users to create profiles, share posts, interact through comments, like posts, follow other users, and receive notifications. The project focuses on core social networking features while providing a clean and responsive user interface.

## Features

* User Registration and Login
* User Profile Management
* Create and Delete Posts
* Like and Unlike Posts
* Comment on Posts
* Follow and Unfollow Users
* Notifications System
* Explore Users Page
* Responsive Design
* Local Storage Data Persistence
